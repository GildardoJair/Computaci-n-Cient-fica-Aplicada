# Computación Científica Aplicada

Este repositorio contiene mis notas personales y de investigación sobre la intersección de la física, las matemáticas y el aprendizaje profundo. El documento está desarrollado íntegramente en **LaTeX** y utiliza **GitHub Actions** para la compilación automática del PDF.

## 📚 Temas Cubiertos
* **Álgebra Lineal:** Espacios vectoriales, transformaciones y descomposición de matrices.
* **Probabilidad:** Fundamentos estocásticos para modelos de datos.
* **Métodos Numéricos:** Solución de ecuaciones y optimización.
* **Deep Learning:** Perceptrones, arquitecturas de redes neuronales y visión computacional.
* **PINNs (Physics-Informed Neural Networks):** Integración de leyes físicas en modelos de aprendizaje automático.

## 🛠️ Tecnologías y Herramientas
* **LaTeX:** Estructura modular, gestión bibliográfica con BibTeX.
* **Python:** Implementaciones de modelos (en la carpeta `/scripts`).
* **Linux Mint:** Entorno de desarrollo principal.
